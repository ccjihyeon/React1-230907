ctrl + `(백틱) = 터미널 열기

node.js 설치확인
    터미널에 "node" 입력
    버전 출력되면 정상 설치 된것.

리액트 설치하는법

    "npm install -g create-react-app" 터미널안에 입력 

리액트 프로젝트 생성 
    "npx create-react-app 프로젝트 이름"

터미널 경로 이동
    cd.. : 상위폴더로 
    cd 폴더이름 : 폴더로진입
    dir : 현재위치에 있는 폴더 목록확인

리액트 내에서 HTML 자동완성
    F1 -> Open setting(JSON)
         "emmet.includeLanguages": { "javascript": "javascriptreact" },

         rfc입력
